Face-Recognition

A Face Recognition project using the ORL Faces Dataset (40 subjects, 10 grayscale images each). Images are converted to vectors, split into training/testing sets, and classified using PCA and LDA with K-NN. The project compares accuracies for different PCA alphas, LDA projections, and K values, with bonus tasks for face vs. non-face classification and alternative train/test splits.
